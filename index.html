uWolf

A Wolfenstein 3D raycasting engine that reads your own game data. No content ships with it.

1
Point it at your data

Put three files from your registered Wolfenstein 3D (v1.4) copy into this page's folder (your web server's Wolfenstein root): VSWAP.WL6, MAPHEAD.WL6, GAMEMAPS.WL6.

Optional, for the original status bar & BJ face: also add VGAGRAPH.WL6, VGAHEAD.WL6, VGADICT.WL6.

The game data loads automatically from this folder. If it isn't found, add the files above and press Retry.

2 · Continue a saved game

Controls

Move: W/A/S/D, turn ←/→. Fire: Ctrl. Weapons: 1–4. Use (doors / secret walls / elevator): Space or E. Map: M.

Touch: left half = move stick, right half = drag to turn, WPN cycles weapons.
